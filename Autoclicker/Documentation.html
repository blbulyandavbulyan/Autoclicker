Документация Autoclicker:

  • Аргументы командной строки(АКС)
  • Горячие клавиши(ГК)
  • Настройка интервала кликов

Аргументы командной строки(АКС):

    Управление треем:

    --flip-to-tray
    Если использовать данный аргумент командной строки то программа при запуске сразу же попадёт в трей.

    Основные настройки программы:

    --click-the-right-mouse-button
    При использовании этого аргумента программа при запуске будет настроена на то, чтобы кликать правой кнопкой мыши.
    --set-miliseconds-of-clicks %interval-of-clicks%
    Устанавливает интервал кликов, %interval-of-clicks% - значение которое будет установлено.
    --set-max-miliseconds %max-interval-of-clicks-value%
    Устанавливает максимальный предел интервала кликов, %max-interval-of-clicks-value% - максимальное значение интервала кликов.

Горячие клавиши(ГК):

    ГК под ИД 0:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+C.
    Данная горячая клавиша запускает и останавливает процесс кликов.
    ГК под ИД 1:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+S.
    Данная горячая клавиша открыват диалоговое окно настроек программы.
    ГК под ИД 2:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+Q.
    Данная горячая клавиша завершает работу программы.
    ГК под ИД 3:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+R.
    Данная горячая клавиша включает режим кликов правой кнопкой мыши.
    ГК под ИД 4:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+E.
    Данная горячая клавиша включает режим кликов левой кнопкой мыши.
    ГК под ИД 5:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+Z.
    Данная горячая клавиша активирует ввод интервала кликов.
    ГК под ИД 6:
    По умолчанию она настроена на комбинацию клавиш Ctrl+Alt+Shift+G.
    Не документирована, нажимать не рекомендуется.
